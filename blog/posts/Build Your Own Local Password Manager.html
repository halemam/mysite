How to Build a Simple Local Password Manager

A real-world, beginner-friendly guide to building your own password manager using Vue.js and Flask. No cloud. No subscriptions. Just something private, useful, and totally yours.

🔒 Why I Built My Own Password Manager

You know those password apps or browser tools everyone uses? They work — but they come with strings attached. They're either paid, limited, or trust you to hand over your entire digital life.

So I built my own. And here’s why you might want to do the same:

👍 What’s Great About Having Your Own:

  • Totally Private – Your passwords never leave your computer.
  • No Subscriptions – You don’t pay anyone to protect your stuff.
  • You Control It – Want a lock screen? Add one.
  • Works Offline – No internet needed.
  • You Learn A Lot – Great project to learn Vue, Python, and encryption.

👎 Why I Stopped Using 3rd-Party Managers:

  • Some were slow or limited unless you paid
  • Some track your data
  • Trusting someone else with your master key isn't ideal

🧰 What You'll Need

  • Vue.js (for the UI)
  • Flask (for backend)
  • SQLite (to save passwords)
  • Python (to run Flask)

🏗 Step 1: Make the Project Folders

mkdir password_manager
cd password_manager

🌐 Step 2: Set Up the Frontend (Vue)

npm install -g @vue/cli
vue create frontend

Choose Manually select features → Babel, Router, SCSS, Linter (Prettier)

cd frontend
npm run serve

Visit: http://localhost:8080

🔒 Step 3: Set Up the Backend (Flask)

mkdir backend
cd backend
python3 -m venv venv
source venv/bin/activate
pip install flask flask-cors python-dotenv cryptography
# .env
SECRET_KEY=your_generated_key_here

Build vault.py for encryption, and app.py for routes.

python app.py

Visit: http://localhost:5000

🔄 Step 4: Connect Frontend to Backend

In Add.vue, create a form to save site name, username, and password using fetch().

In Vault.vue, use fetch() to view saved entries and add delete buttons.

💻 One-Click Script (KDE / Konsole)

# start.sh
#!/bin/bash
konsole --new-tab -e bash -c "cd backend; source venv/bin/activate; python app.py; exec bash" &
konsole --new-tab -e bash -c "cd frontend; npm run serve; exec bash" &
chmod +x start.sh
./start.sh

✅ That’s It!

  • Runs on your own computer
  • Passwords are encrypted
  • Looks clean and works fast
