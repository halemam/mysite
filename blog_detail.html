How I Built My Website with GitHub Pages

Published April 2025

I started off building my site in Django — templates, views, and the full backend. But I realized most of what I needed was just clean, fast HTML with a nice design.

So I converted everything into static pages, hosted it on GitHub Pages for free, and kept the futuristic glow and animations I liked. No more server dependencies. Just vibes and clean code.

This page you're reading now is built with love, CSS, and a touch of rebellion. 🚀
